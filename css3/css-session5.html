Pseudo Elements

First Letter, First Line, Selection

Pseudo-elements in CSS are used to style specific parts of an element — not the whole element. They let you insert or style content that isn’t in the HTML directly.

Before, After, Content

::before

  • Inserts content before the actual content of an element.
  • Must include the content property (even if empty).

::after

  • Inserts content after the actual content of an element.
  • Also requires the content property.

content property

The content property defines what to display inside the pseudo-element.
