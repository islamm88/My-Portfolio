Introduction

what is JavaScript ?

  • JavaScript (often abbreviated as JS) is a high-level, interpreted programming language that is most commonly used to make web pages interactive and dynamic.
  • Client-side scripting language: Runs directly in the userâ€™s web browser without needing to be compiled.
  • Core web technology: Alongside HTML (structure) and CSS (style), JavaScript adds behavior to web pages.
  • Interactivity: Used for things like dropdown menus, form validation, animations, image sliders, and interactive maps.
  • Server-side: While originally designed for the browser, JavaScript can now run on servers (e.g., Node.js) and even in desktop and mobile apps.
  • JavaScript : can update and change both HTML and CSS.
