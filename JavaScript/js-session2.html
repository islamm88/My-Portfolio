Arithmetic Operators

Arithmetic Operations in JavaScript

Arithmetic operations are the basic math operations you can use in JS such as:

  • Addition : let x = a + b;
  • Subtraction : let z = x - y;
  • Multiplication : let z = x * y;
  • Division : let z = x / y;
  • Modulus : let z = x % y;
  • Exponent : let z = x ** 2;
  • Increment : x++;
  • Decrement : x--;

Operation Priority (Order of execution)

  1. Parentheses ()
  2. Exponents **
  3. Multiplication / Division * /
  4. Addition / Subtraction + -
