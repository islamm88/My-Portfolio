Arithmetic Operators

Arithmetic Operations in JavaScript

Arithmetic operations are the basic math operations you can use in JS such as:

  • Addition : let x = a + b;
  • Subtraction : let z = x - y;
  • Multiplication : let z = x * y;
  • Division : let z = x / y;
  • Modulus : let z = x % y;
  • Exponent : let z = x ** 2;
  • Increment : x++;
  • Decrement : x--;

Operation Priority (Order of execution)

  1. Parentheses ()
  2. Exponents **
  3. Multiplication / Division * /
  4. Addition / Subtraction + -

Conditional Statements

  • Conditional Statements allow us to perform different actions for different conditions.
  • Conditional statements run different code depending on true or false conditions.
  • Conditional statements include:
    • if
    • if...else
    • if...else if...else
    • switch
    • ternary (? :)

if statement in JavaScript

The if statement is used to make a decision in your program.

It checks a condition.

  • If the condition is true , the code inside runs
  • If it’s false, the code is skipped

if

The condition must return true or false.

if ... else

How it works (step by step)

  • JavaScript checks the condition inside if
  • JavaScript checks the condition inside if
