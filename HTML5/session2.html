Anchor

In HTML, an anchor is created using the <a> tag and is used to Create hyperlinks (link to another page, section, file, or website) , Create anchors within the same page to jump to a specific section.

How to Type anchor tag

<a href = "https://www.example.com" target = "_blank" >Visit Example</a>

anchor has two element attributes & clickable text

  • href = " "
  • target = " "
  • Text between the tags is the clickable link like Google.
