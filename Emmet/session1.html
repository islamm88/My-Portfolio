Introduction

Emmet is a powerful code snippet tool that helps web developers write HTML and CSS faster by using abbreviations that automatically expand into full code.


What Emmet Does in HTML:

Instead of writing long lines of HTML manually, Emmet lets you type short expressions and expand them into complete HTML tags.

Example

To Create div by class or id .class name using dot
#id name using hash
.header
#header
To Create element by class or id p.class name using dot
p#id name using hash
p.pr1
p#pr2
Child: > You can use > operator to nest elements inside each other:
header>.logo
Sibling: + Use + operator to place elements near each other, on the same level:
header>.logo+nav
Multiplication: * With * operator you can define how many times element should be outputted:
header>.logo+nav>ul>li*5
Climb-up: ^ With ^ operator, you can climb one level up the tree and change context where following elements should appear:
div+div>p>span+em^img
Grouping: () Parenthesises are used by Emmetsâ€™ power users for grouping subtrees in complex abbreviations:
div>(header>ul>li*2>a)+footer>p
ID and CLASS In CSS, you use elem#id and elem.class notation to reach the elements with specified id or class attributes. In Emmet, you can use the very same syntax to add these attributes to specified element:
div#header+div.page+div#footer.class1.class2.class3
