# Bootstrap
## Download Bootstrap

## CDN
    Use Bootstrap directly from a CDN without installing any files:

    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css" rel="stylesheet">
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/js/bootstrap.bundle.min.js"></script>

## Reboot in Bootstrap

    Bootstrap Reboot is a foundational CSS layer included in Bootstrap.
    Its purpose is to normalize and improve default browser styles so elements look consistent across browsers.
