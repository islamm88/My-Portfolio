Introduction

Bootstrap

  • frontend toolkit .
  • Build fast .
  • responsive sites .
  • Currently v5.3.8
  • Build and customize with Sass

A Frontend Toolkit is a set of tools or libraries that help you build website and application interfaces quickly and efficiently.

Important globals

  • Bootstrap requires the use of the HTML5 doctype <!DOCTYPE html> . Without it, you’ll see some funky and incomplete styling.
  • Bootstrap is developed mobile first so necessary using CSS media queries
  • you need to override box-sizing if you find problem with third party like Google Maps and Google Custom Search Engine. .selector-for-some-widget { box-sizing: content-box; }
  • Bootstrap Reboot For improved cross-browser rendering .

What is Reboot in Bootstrap?

Bootstrap Reboot is a small CSS file included in Bootstrap.

Its job is to normalize and clean up the default styles of HTML elements across all browsers.

Reboot works automatically when you include Bootstrap.

Getting started

Quick start

  1. Create a new index.html file in your project root. Include the <meta name="viewport"> tag as well for proper responsive behavior in mobile devices.
    • <meta name="viewport" content="width=device-width, initial-scale=1">
  2. Include Bootstrap’s CSS and JS.
    • <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-sRIl4kxILFvY47J16cr9ZwB07vP4J8+LH7qKQnuqkuIAvNWLzeN8tE5YBujZqJLB" crossorigin="anonymous">
    • <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/js/bootstrap.bundle.min.js" integrity="sha384-FKyoEForCGlyvwx9Hj09JcYn3nv7wiPVlz7YYwJrWVcXK/BmnVDxM+D2scQbITxI" crossorigin="anonymous" ></script >

Download Bootstrap

Download Bootstrap (Local Files)

  1. Go to the official Bootstrap website https://getbootstrap.com
  2. Click Download
  3. Choose Compiled CSS and JS
  4. Download the ZIP file
  5. Extract it into your project folder

File References

In your project, you will need to reference the following Bootstrap files:

  • From CSS: bootstrap.css and bootstrap.min.css
  • From JS: bootstrap.js and bootstrap.min.js

you can include bootstrap.min.css & bootstrap.min.js in your HTML file like this:

  • <link href="path/to/bootstrap.min.css" rel="stylesheet">
  • <script src="path/to/bootstrap.min.js"></script>

Margin & Padding

Breakpoints

What are Breakpoints in Bootstrap?

Breakpoints are screen width points where the layout changes to fit different devices.

Bootstrap uses breakpoints to make your website responsive.

Breakpoint Class infix Dimensions
Extra small None < 576px
Small sm ≥ 576px
Medium md ≥ 768px
Large lg ≥ 992px
Extra large xl ≥ 1200px
Extra extra large xxl ≥ 1400px

Padding & Margin

Padding and (Paddding Responsive)

p{direction}-{size}

  • (( Direction ))
    • .p Padding
    • .pt padding-top
    • .pb padding-bottom
    • .ps padding-start
    • .pe padding-end
    • .px padding (left & Right)
    • .py padding (up & Down)
  • (( Size ))
    • 0 = 0
    • 1 = 0.25rem
    • 2 = 0.5rem
    • 3 = 1rem
    • 4 = 1.5rem
    • 5 = 3rem
